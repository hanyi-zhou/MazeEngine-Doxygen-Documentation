Detailed Description

Module to create destination game entities for the maze game.

This module provides a function to create destination game entities for the maze game.

Attributes:
    STATIC_TEXTURE (str): Path to the static texture for the destination.
    DX (float): Offset in the x-direction from the given position for the destination.
    DY (float): Offset in the y-direction from the given position for the destination.
    TRANSFORM_WIDTH (float): Width of the destination's transform.
    TRANSFORM_HEIGHT (float): Height of the destination's transform.
