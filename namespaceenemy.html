Detailed Description

Module to create enemy game entities for the maze game.

This module provides a function to create enemy game entities for the maze game.

Attributes:
    ANIMATION_TEXTURES (list): List of paths to the animation textures for the enemy.
    FRAME_RATE (float): Frame rate of the enemy's animation.
    DX (float): Offset in the x-direction from the given position for the enemy.
    DY (float): Offset in the y-direction from the given position for the enemy.
    TRANSFORM_WIDTH (float): Width of the enemy's transform.
    TRANSFORM_HEIGHT (float): Height of the enemy's transform.
