Detailed Description

A maze level editor to create maze levels for a maze game.

Attributes:
    root (tk.Tk): The root window of the application.
    canvas_width (int): The width of the canvas.
    canvas_height (int): The height of the canvas.
    pixel_size (int): The size of each pixel in the canvas grid.
    last_x (int): The x-coordinate of the last mouse event.
    last_y (int): The y-coordinate of the last mouse event.
    draw_option (str): The current drawing option selected by the user.
    main_character_placed (bool): Indicates whether the main character is placed.
    destination_placed (bool): Indicates whether the destination is placed.
    main_character_image (ImageTk.PhotoImage): The image of the main character.
    destination_image (ImageTk.PhotoImage): The image of the destination.
    enemy_image (ImageTk.PhotoImage): The image of the enemy.
    jewel_image (ImageTk.PhotoImage): The image of the jewel.
    maze_data (dict): A dictionary to store maze data for different levels.
    current_level (int): The current level being edited.
