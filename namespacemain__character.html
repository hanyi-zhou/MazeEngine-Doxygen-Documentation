Detailed Description

Module to create the main character game entity for the maze game.

This module provides a function to create the main character game entity for the maze game.

Attributes:
    LEFT_ANIMATION_TEXTURES (list): List of paths to the animation textures for the main character when moving left.
    RIGHT_ANIMATION_TEXTURES (list): List of paths to the animation textures for the main character when moving right.
    UP_ANIMATION_TEXTURES (list): List of paths to the animation textures for the main character when moving up.
    DOWN_ANIMATION_TEXTURES (list): List of paths to the animation textures for the main character when moving down.
    FRAME_RATE (float): Frame rate of the main character's animation.
    DX (float): Offset in the x-direction from the given position for the main character.
    DY (float): Offset in the y-direction from the given position for the main character.
    TRANSFORM_WIDTH (float): Width of the main character's transform.
    TRANSFORM_HEIGHT (float): Height of the main character's transform.
    SPEED (float): Speed of the main character.
    INITIAL_DIRECTION (Direction): Initial direction of the main character.
