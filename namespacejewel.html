Detailed Description

Module to create jewel game entities for the maze game.

This module provides a function to create jewel game entities for the maze game.

Attributes:
    ANIMATION_TEXTURES (list): List of paths to the animation textures for the jewel.
    FRAME_RATE (float): Frame rate of the jewel's animation.
    DX (float): Offset in the x-direction from the given position for the jewel.
    DY (float): Offset in the y-direction from the given position for the jewel.
    TRANSFORM_WIDTH (float): Width of the jewel's transform.
    TRANSFORM_HEIGHT (float): Height of the jewel's transform.
